요약

최근 공식 업데이트는 단순 코드 생성보다 장기 실행 에이전트 운영에 초점이 있다. Claude는 1M 컨텍스트와 거절/대체 모델 정책을 포함한 Fable 5 계열을 공개했고, Cursor는 화면 선택과 SDK auto-review를 강화했으며, Codex는 Desktop handoff, 플러그인 JSON, 이미지 파일 경로 노출 같은 운영 기능을 안정화했다.

Claude Code / Anthropic

Claude Fable 5와 Mythos 5 공개

확인 완료1M context안전 정책 중요

Claude Platform 릴리스 노트는 2026-06-09에 Claude Fable 5와 Project Glasswing 참여자용 Claude Mythos 5를 공개했다고 적고 있다. 두 모델은 기본 1M 토큰 컨텍스트, 128k 최대 출력, adaptive thinking을 지원한다. Fable 5는 요청 및 생성 중 안전 classifier를 실행하며, 출력 전 refusal은 과금되지 않는다고 명시됐다.

Cursor

Cursor 3.7: Design Mode Improvements

확인 완료frontend workflow

Cursor 3.7은 브라우저 Design Mode에서 여러 UI 요소를 동시에 선택하고, 음성으로 다음 변경을 큐잉하는 기능을 강조한다. 에이전트가 코드뿐 아니라 화면상 선택 요소, 주변 레이아웃, 시각 관계를 함께 보게 되는 점이 프론트엔드 반복 작업에 중요하다.

발표일: 2026-06-05 | 출처: Cursor changelog | 확인 상태: 확인 완료 | URL: https://cursor.com/changelog

Cursor SDK: custom tools, auto-review, JSONL store

확인 완료SDKauditability

2026-06-04 업데이트는 custom tools를 local agent에 함수 정의로 연결하고, headless 실행의 tool call을 auto-review로 라우팅하며, SQLite 대신 JSONL 기반 실행 메타데이터 저장을 선택할 수 있게 했다. CI와 내부 자동화에서 실행 추적과 재현성이 좋아지는 변화다.

발표일: 2026-06-04 | 출처: Cursor changelog | 확인 상태: 확인 완료 | URL: https://cursor.com/changelog

Codex 역할별 플러그인, Sites, annotations

확인 완료pluginsknowledge work

OpenAI는 2026-06-02 Codex를 역할별 업무 도구로 확장한다고 발표했다. 발표문은 주간 사용자 500만 명 이상, 비개발자 사용자가 전체의 약 20%, 역할별 플러그인 6종과 62개 앱, 110개 skills, Business/Enterprise 대상 Sites preview와 annotations를 핵심 변화로 제시한다.

Windows Computer Use와 remote control

확인 완료Windows지역 제한

OpenAI Help Center 릴리스 노트는 2026-05-29 Codex app에서 Windows Computer Use와 remote control을 지원한다고 설명한다. Windows 호스트가 프로젝트 파일, shell, app server, local context를 유지하고, 사용자는 iOS/Android 또는 Mac에서 진행 상황을 확인하고 조정할 수 있다. EEA, UK, Switzerland에서는 출시 시점에 사용할 수 없다.
