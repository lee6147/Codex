주요 저장소

DeusData / codebase-memory-mcp

9,086 stars +4,212 this week MCP

코드베이스 맥락을 지식 그래프처럼 축적하는 MCP 서버다. 장기 세션 품질과 검색 품질을 동시에 보강하는 방향이라 Codex나 Cursor 같은 에이전트 런타임과 직접 맞닿아 있다.

실무 해석: "컨텍스트를 오래 들고 가는 인프라"가 별도의 제품 카테고리로 굳고 있다.

chopratejas / headroom

41,226 stars +12,793 this week 토큰 절감

툴 출력, RAG 조각, 로그를 압축해 컨텍스트 비용을 줄이는 도구다. 에이전트 실사용 비용과 응답 밀도를 동시에 건드리는 주제라 폭발적으로 주목받았다.

실무 해석: 모델 교체보다 입력 압축이 더 즉각적인 ROI를 주는 팀이 많다는 신호다.

Panniantong / Agent-Reach

35,658 stars +8,324 this week 검색/브라우징

실무 해석: 단일 IDE 안의 코드 생성보다 "도구 사용 범위"가 생산성 차이를 만든다.

NVIDIA / SkillSpector

8,597 stars +5,026 this week 보안

실무 해석: 스킬과 자동화가 늘수록 "무엇을 실행시키는가"가 새로운 보안 경계가 된다.

addyosmani / agent-skills

64,022 stars +7,170 this week 스킬 컬렉션

실무 해석: 모델보다 운영 패턴과 재사용 가능한 작업 단위가 경쟁력으로 이동한다.

google-research / timesfm

24,399 stars +3,335 this week foundation model

실무 해석: 에이전트 인프라 열풍 속에서도 강한 기반 모델 자산은 계속 관심을 받는다.

20,025 stars +3,025 this week PM workflow

실무 해석: 에이전트는 개발팀 전용에서 업무 기능별 운영 체계로 넓어지고 있다.
