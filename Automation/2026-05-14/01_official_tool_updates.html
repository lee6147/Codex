Windows 및 백그라운드 안정성 수정

확인 완료2026-05-12

gh 같은 실행 파일 누락 시 Windows에서 반복적으로 동기 where.exe 재실행이 걸리던 이벤트 루프 stall을 고쳤다. claude --bg의 백그라운드 서비스 연결 오류도 수정됐다.

실무 의미: Windows 기반 Codex/OMX 사용자에게도 직접 영향을 주는 안정성 개선이다.

Cursor

클라우드 에이전트 개발 환경

확인 완료2026-05-13

멀티 레포 환경, Dockerfile 기반 환경 구성, 빌드 시크릿, 레이어 캐시, 환경 버전 히스토리와 감사 로그를 도입했다.
