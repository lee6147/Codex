Claude Code

공식 changelog 확인 불가: 날짜

Anthropic의 Claude Code 릴리스 노트는 GitHub changelog로 연결되며, 최신 항목에는 2.1.150, 2.1.149, 2.1.148 등이 표시된다. 2.1.149 항목은 사용량 분류, diff 상세 보기 키보드 스크롤, Markdown task list 렌더링, PowerShell 권한 우회 및 worktree sandbox allowlist 수정 등 운영 안정성 중심의 변경이 많다.

다만 해당 changelog 본문에는 각 버전의 게시일이 명시되어 있지 않아, 2026-05-24~2026-05-27 기간 내 공식 업데이트라고 확정하지 않았다.

Cursor

가장 가까운 공식 항목은 2026-05-20 Cursor 3.5

공식 changelog 기간 외 참고

Cursor 공식 changelog의 최신 주요 항목은 2026-05-20의 3.5 업데이트다. Shared Canvases, Cursor Automations의 Agents Window 통합, multi-repo automations, no-repo automations, 신규 템플릿이 핵심이다.

실사용 관점에서는 Cursor가 단순 IDE 보조 도구에서 팀 단위 자동화 운영면으로 이동하고 있다는 점이 중요하다. 특히 저장소가 없거나 여러 저장소를 물고 가는 automation은 기존 코드 편집기보다 운영 에이전트에 가까운 사용성을 만든다.
