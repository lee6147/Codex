패턴 분석

패턴관련 저장소의미
에이전트 운영체계화 compound-engineering-plugin, revfactory/harness, babysitter 도구가 코드를 쓰는 단계를 넘어 역할, 권한, 반복 실행, 검증을 포괄하는 workflow layer로 이동한다.
컨텍스트/메모리 인프라 supermemory, codegraph 긴 작업에서 핵심 병목은 모델 성능만이 아니라 기억, 코드 구조 인덱싱, 검색 품질이다.
문서 변환과 지식 흡수 markitdown 업무 문서가 agent-readable Markdown으로 바뀌면 코드 작업과 문서 작업의 경계가 낮아진다.
플러그인 생태계 cursor/plugins, anthropics/claude-code 각 제품이 extension, skill, plugin으로 반복 가능한 작업 단위를 공식화하고 있다.

  • 단순 star 수보다 “어떤 agent surface와 연결되는가”를 봐야 한다. Claude Code, Codex, Cursor를 동시에 언급하는 저장소는 adoption 신호가 강하다.

  • 이번 주는 AI 앱 생성 자체보다 agent를 제어하고 기억시키고 검토하는 보조 시스템이 더 눈에 띈다.
