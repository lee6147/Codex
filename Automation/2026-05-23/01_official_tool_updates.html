Claude Code

확인 완료2026-05-22

v2.1.148: Bash 도구 회귀 수정

Anthropic의 GitHub 릴리스는 2.1.148을 최신 버전으로 표시하며, 2.1.147에서 일부 사용자에게 모든 Bash 명령이 exit code 127로 반환되던 회귀를 수정했다고 밝혔다.

실무 의미: 2.1.147을 즉시 도입한 팀은 shell 기반 검증과 빌드가 실패처럼 보일 수 있었으므로 2.1.148로 올리는 것이 안전하다.

Claude Code v2.1.148 릴리스

확인 완료2026-05-21

v2.1.147: 배경 세션, 코드 리뷰, Windows 안정성

v2.1.147은 pinned background session을 유휴 상태에서도 유지하고, 업데이트 적용 시 제자리에서 재시작하게 했다. `/simplify`는 `/code-review`로 전환되어 correctness bug 중심의 리뷰와 GitHub PR inline comment 옵션을 제공한다.

실무 의미: 장시간 agent 작업과 PR 리뷰 자동화 쪽 기능이 명확해졌다. Windows 사용자는 PowerShell 출력 누락, winget/Microsoft Store pwsh 실행 실패, 권한 재사용 규칙 같은 문제 수정도 함께 받는다.

Cursor changelog

근접 공식 항목2026-05-20

Automations in Agents Window

Cursor는 Automations를 Agents Window에서 만들고 관리할 수 있게 했고, 여러 repo를 붙이거나 repo 없이 automation을 구성할 수 있게 했다. 새 automation agent run에는 7일간 50% 할인도 공지했다.

실무 의미: Cursor는 IDE 안 agent 실행에서 반복 작업 스케줄링과 업무 도구 연결로 무게중심을 넓히고 있다.

OpenAI Codex

확인 완료2026-05-21

Codex app 26.519: Appshots, Goal mode, remote computer use

OpenAI Codex changelog는 Appshots를 macOS Codex app에 추가했고, Goal mode를 실험 기능에서 일반 기능으로 올렸다고 밝혔다. 또한 Mac 잠금 이후에도 Codex Mobile 등을 통해 데스크톱 앱을 사용할 수 있는 remote computer use, Business용 plugin sharing, in-app browser annotation과 브라우저 안정성 개선을 포함했다.

실무 의미: Codex는 단발성 코드 작성보다 장시간 목표 실행, 다른 앱 맥락 활용, plugin/skill 공유로 운영 범위를 넓히는 중이다.

Codex changelog

확인 완료2026-05-21

Codex CLI 0.133.0: goal 저장, 권한 프로필, plugin discovery

CLI 0.133.0은 goals를 기본 활성화하고 전용 저장소와 turn 간 progress tracking을 붙였다. remote-control은 준비 상태와 machine status를 더 명확히 보고하며, permission profile은 list API, inheritance, managed requirements, Windows sandbox 통합이 보강됐다.

실무 의미: CLI 자동화에서 상태 추적, 권한 정책, plugin 목록 가시성이 중요해진다. 장시간 agent 목표를 운영하려는 팀은 0.133.0 변경을 우선 확인해야 한다.
