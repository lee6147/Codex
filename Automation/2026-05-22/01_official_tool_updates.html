Claude Code

확인 완료2026-05-21

v2.1.146: 코드 리뷰 명령과 Windows 안정성 개선

Anthropic의 GitHub 릴리스에 따르면 `/simplify`가 `/code-review`로 이름이 바뀌었고, effort level 지정이 가능해졌다. Auto mode가 명시적 질문 요청을 억제하지 않도록 수정됐으며, winget 또는 Microsoft Store로 설치된 `pwsh`에서 PowerShell 도구가 실패하던 회귀도 고쳤다.

실무 의미: Windows 기반 자동화와 다중 에이전트 운용에서 막히던 지점들이 직접 수정됐다. Claude Code를 상태 표시줄, MCP 서버, 백그라운드 세션과 엮어 쓰는 팀에는 즉시 업데이트 가치가 있다.

확인 완료2026-05-19

v2.1.145: 에이전트 JSON, 추적성, 플러그인 탐색 강화

`claude agents --json`으로 실행 중인 세션을 스크립트에서 읽을 수 있게 됐고, OTEL span에 agent id 계층 정보가 추가됐다. 플러그인 설치 전 명령, 에이전트, 스킬, hook, MCP/LSP 서버를 보여주는 개선도 포함됐다.

실무 의미: Claude Code가 단일 CLI를 넘어 상태 감시, 세션 선택기, 팀 대시보드와 붙는 운영 표면으로 확장되고 있다.

Agent view, goal 지속 실행, Opus 4.7 fast mode

Cursor

2026-05-20~2026-05-22 공식 AI 코딩 업데이트 부재

Cursor 공식 changelog에서 엄격한 3일 창에 해당하는 새 AI 코딩 업데이트는 확인하지 못했다. 이 항목은 확인 불가가 아니라 "기간 내 없음"으로 기록한다.

실무 의미: 오늘의 Cursor 판단은 새 릴리스보다는 직전 릴리스의 방향성, 특히 Jira와 cloud agent 확장에 근거해야 한다.

근접 공식 항목2026-05-18

Cursor in Jira 및 Composer 2.5

Cursor는 Jira work item에서 `@Cursor`를 호출해 cloud agent를 시작하고 PR 링크를 남기는 흐름을 공개했다. 같은 날 Composer 2.5도 공개되며 장기 작업 지시 준수와 협업 품질 개선을 강조했다.

실무 의미: Cursor의 초점은 에디터 보조에서 이슈 트래커 기반 작업 할당과 팀 운영으로 넓어지고 있다.

근접 공식 항목2026-05-13

Agents Window, compact chat, multi-repo environments

Cursor 3.4는 Agents Window의 전체 화면 탭, compact chat, PR 탭 개선, cloud agent 상태와 환경 선택 안정성을 다뤘다. Cloud agents와 automations의 multi-repo environment도 공식적으로 강조됐다.

실무 의미: 여러 저장소를 묶어 에이전트 실행 환경을 재사용하는 팀에는 Cursor의 관리형 agent 환경이 점점 중요한 차별점이다.

OpenAI Codex

3일 창 내 Codex 전용 새 항목 없음

공식 릴리스 노트에서 Codex 새 항목 미확인

OpenAI Help Center의 Enterprise/Edu 릴리스 노트는 2026-05-18에 Apps Directory beta label 제거를 기록했지만, Codex 전용 기능 업데이트는 아니었다. 2026-05-20~2026-05-22 범위에서도 Codex 전용 공식 항목은 확인하지 못했다.

실무 의미: 오늘 Codex 평가는 새 기능보다는 2026-05-14 원격 접속과 access token 발표의 후속 운영 영향에 초점을 맞춘다.

근접 공식 항목2026-05-14

ChatGPT 모바일 앱을 통한 Codex remote access

OpenAI 공식 릴리스 노트는 Codex가 ChatGPT 모바일 앱에서 원격 접속을 지원한다고 설명한다. 사용자는 장기 실행 작업 상태, 승인, 스크린샷, 터미널 출력, diff, 테스트 결과를 확인하고 개입할 수 있다.

실무 의미: Codex는 데스크톱 앱만의 작업 큐가 아니라, 장시간 실행되는 로컬 또는 원격 작업을 모바일에서 감독하는 운영 체계로 확장 중이다.

근접 공식 항목2026-05-14

Enterprise access token for automation

OpenAI는 Enterprise workspaces에서 브라우저 로그인 없이 신뢰된 비대화형 로컬 워크플로가 ChatGPT workspace identity와 enterprise controls를 사용할 수 있는 Codex access token을 제공한다고 밝혔다.

실무 의미: 반복 자동화, 야간 빌드, 내부 agent harness에서 사람의 브라우저 세션 의존도를 낮출 수 있다. 다만 관리자 정책과 governance 표면 확인이 선행돼야 한다.
