Cursor

Cursor 3.9 (2026-06-22): Customize Cursor로 plugins, skills, MCPs를 한 화면에 통합

공식 changelog Customize Marketplace
  • 새 Customize 페이지에서 plugins, skills, MCPs, subagents, rules, commands, hooks를 user·team·workspace 레벨로 추가하고 관리할 수 있다.

Cursor 3.8 (2026-06-18): Automations를 GitHub·Slack·computer use까지 확장

공식 changelog Automations GitHub triggers
  • /automate로 로컬 agent 세션 안에서 바로 automation을 만들 수 있고, Slack emoji trigger와 5종의 GitHub trigger가 추가됐다.
  • cloud agents가 automation에서 computer use를 기본으로 쓸 수 있게 되어 데모 제작과 UI 산출물 생성까지 자동화 범위가 넓어졌다.
  • 미완성 automation 저장, 기본 PR 열기, memory file 삭제 같은 운영 세부 기능도 보강돼 Cursor의 중심축이 editor 보조에서 workflow runtime으로 더 이동하고 있다.

OpenAI Codex

Mastering Codex Remote for engineering (2026-06-23): 휴대폰을 제어 평면으로 쓰는 운영 패턴 정리

공식 블로그 Remote Control plane
  • OpenAI는 휴대폰을 작은 터미널로 복제하는 대신, 시작·steer·review·organize를 담당하는 제어 평면으로 쓰는 것이 Codex Remote의 핵심이라고 정리했다.
  • 최근 두 달 사이 remote host connections, worktrees, goals, side chats, inline code review, queued/steering prompts, attachments, skills, plugins, archived threads, security controls가 누적됐다고 명시했다.
  • 이 글은 Codex가 단순 코드 생성기가 아니라 장시간 엔지니어링 운영을 분산 제어하는 인터페이스로 포지셔닝되고 있음을 보여준다.

발표일: 2026-06-23 | 출처: OpenAI Developers Blog | 링크: https://developers.openai.com/blog/mastering-codex-remote-for-engineering

Codex Mobile / App 6월 흐름: 진행 가시성, autocomplete, Record & Replay, host handoff 축 강화

공식 changelog Codex Mobile App 26.616
  • 2026-06-22 ChatGPT for iOS 1.2026.167은 commands·skills·plugins autocomplete, subagent/task/worktree 진행 가시성, workspace file search와 code review drafts를 개선했다.
