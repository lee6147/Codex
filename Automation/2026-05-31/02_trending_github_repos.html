microsoft/markitdown

2,473 stars today

Office 문서와 파일을 Markdown으로 변환하는 Python 도구입니다. AI agent가 문서와 레거시 파일을 읽기 쉬운 텍스트 컨텍스트로 바꾸는 전처리 단계에 바로 연결됩니다.

주목 포인트: RAG, 에이전트 리서치, 문서 기반 코딩 작업에서 "파일을 Markdown으로 정규화"하는 수요가 계속 커지고 있습니다.

harry0703/MoneyPrinterTurbo

2,775 stars today

LLM을 이용해 짧은 영상을 생성하는 자동화 프로젝트입니다. 코딩 도구 자체는 아니지만, agent가 스크립트·소스·렌더링 파이프라인을 묶는 대표적인 end-to-end 자동화 예시입니다.

주목 포인트: 멀티모달 콘텐츠 생성 자동화는 단일 모델 성능보다 워크플로 안정성과 검수 단계가 경쟁력이 됩니다.

주목 포인트: AI 코딩 도구 경쟁축이 "질문 답변"에서 "코드베이스를 읽고 명령 실행, Git workflow 처리"로 이동했음을 보여줍니다.

주목 포인트: 플러그인은 단순 확장 기능이 아니라 agent에게 허용할 도구, 컨텍스트, 실행 경계를 정의하는 운영 단위가 되고 있습니다.

주목 포인트: "모델에게 시키기"보다 "팀/역할/검증 절차를 설계하기"가 생산성 차이를 만드는 영역으로 부상 중입니다.

주목 포인트: 특정 IDE/CLI 하나에 종속되지 않는 agent workflow layer에 대한 수요를 보여줍니다.
