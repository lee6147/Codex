공식 AI 코딩 도구 업데이트

생성일: 2026-06-15 · 조사 범위: 2026-06-12 ~ 2026-06-15 우선, 공식 최신 항목 보강 포함

요약

최근 3일 안에 세 도구 모두에서 명확한 날짜가 붙은 대형 공식 릴리스가 충분히 나오지는 않았다. 확인 가능한 공식 자료 기준으로는 OpenAI Codex가 2026-06-11 앱 업데이트를 냈고, Cursor는 2026-06-10 Bugbot 성능 개선을 공지했으며, Claude Code는 GitHub CHANGELOG의 최신 항목이 빠르게 누적되고 있으나 항목별 게시일은 원문에서 확인되지 않는다.

이번 주의 실무 신호는 모델 자체 발표보다 운영 품질이다. 리뷰 자동화, 브라우저 디버깅, 권한/모델 제한, 백그라운드 세션 안정성이 공통 개선축으로 보인다.

Claude Code

공식 CHANGELOG확인 상태: 부분 확인

2.1.176: 세션 제목, 모델 제한, 원격 제어, 백그라운드 안정성

발표일: 확인 불가. 원문 CHANGELOG가 버전 순으로 최신 항목을 제공하지만 항목별 날짜를 노출하지 않는다.

핵심: 대화 언어 기반 세션 제목 생성, Bedrock 자격 증명 캐싱 개선, availableModels 강제 적용 보완, Remote Control 연결/모델 전환 문제 수정, Windows 백그라운드 서비스와 tmux 복사 문제 수정이 포함됐다.

실무 의미: 대규모 조직에서 모델 allowlist와 원격/백그라운드 세션을 쓰는 경우 예측 불가능한 모델 전환과 세션 복구 문제를 줄이는 업데이트다.

원문 CHANGELOG

공식 CHANGELOG확인 상태: 부분 확인

2.1.175 ~ 2.1.172: 관리형 모델 정책과 중첩 서브에이전트

발표일: 확인 불가.

핵심: enforceAvailableModels 관리 설정, 모델 피커 표시 수정, sub-agent가 최대 5단계까지 sub-agent를 생성하는 기능, 플러그인 탐색 검색, 권한 패턴 매칭 수정이 확인된다.

실무 의미: Claude Code가 단일 터미널 보조 도구에서 조직 정책과 다중 에이전트 운영을 받는 실행 환경으로 이동하고 있음을 보여준다.

Cursor

공식 changelog확인 완료

Bugbot이 3배 이상 빨라지고 비용은 22% 감소

발표일: 2026-06-10

핵심: 평균 리뷰 시간이 약 5분에서 약 90초로 줄고, 리뷰당 발견 버그 수가 평균 0.56개에서 0.62개로 증가했으며, 실행 비용은 약 22% 낮아졌다고 Cursor가 공지했다.

실무 의미: AI가 작성한 PR을 별도 리뷰 에이전트로 다시 거르는 흐름이 비용/속도 면에서 더 현실적인 기본 옵션이 되고 있다.

Cursor changelog

공식 changelog확인 완료

/review로 push 전 Bugbot 및 Security Review 실행

발표일: 2026-06-10

핵심: Cursor 3.7+와 cursor.com/agents에서 push 전에 /review, /review-bugbot, /review-security를 실행할 수 있다. 같은 diff로 PR을 열면 GitHub/GitLab의 Bugbot이 이미 검토된 diff로 인식한다.

실무 의미: 리뷰가 PR 생성 후 이벤트가 아니라 로컬 변경 단계의 반복 검증 루프로 이동한다.

OpenAI Codex

공식 changelog확인 완료

Codex app 26.609: Developer mode, /init, rate-limit reset banking

발표일: 2026-06-11

핵심: Browser use용 Developer mode가 Chrome 및 Codex 인앱 브라우저에 추가되어 CDP 기반 성능 프로파일링, 네트워크/콘솔/런타임 오류 디버깅을 지원한다. 앱 composer에 /init 명령도 추가됐다.

실무 의미: Codex가 코드 편집뿐 아니라 브라우저 기반 프런트엔드 진단, 프로젝트 지침 초기화, 사용량 관리까지 한 작업면에 통합하고 있다.

확인 불가 및 해석

  • Claude Code 공식 CHANGELOG는 최신 버전 내용을 제공하지만 항목별 게시일이 원문에 없어 발표일은 확인 불가로 표시했다.
  • Cursor와 Codex의 최신 공식 대형 항목은 각각 2026-06-10, 2026-06-11로, 2026-06-15 기준 엄격한 최근 3일 창에는 걸치지 않는다. 다만 이번 주 최신 공식 근거로 보강했다.
  • 공식 업데이트는 세 제품 모두 “더 강한 모델 발표”보다 권한, 리뷰, 브라우저 디버깅, 백그라운드/원격 세션 안정화에 집중되어 있다.
