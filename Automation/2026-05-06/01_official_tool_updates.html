Cursor

Model controls, spend management, and usage analytics

발표일 2026-05-04 Cursor Changelog 확인 완료

기능 또는 업데이트명: Model controls, spend management, and usage analytics

핵심 설명: Enterprise 관리자는 이제 모델/제공자 단위 allowlist·blocklist를 더 세밀하게 다룰 수 있고, 새로운 제공자나 모델 버전을 기본 차단하도록 설정할 수 있다. 또한 hard limit 대신 soft limit를 두고 50%·80%·100% 자동 경고를 보내며, usage analytics는 clients, Cloud Agents, automations, Bugbot, Security Review 단위로 분해된다.

실사용 관점에서 중요한 이유: Cursor의 경쟁력은 더 이상 “좋은 AI IDE”에만 있지 않다. 모델 승인 정책, 소비량 경보, 제품 surface별 usage 해석이 붙으면서 조직 운영 툴로 성격이 더 강해졌다.

기능 또는 업데이트명: Using Codex with your ChatGPT plan 문서 갱신

핵심 설명: 현재 문서는 Codex가 ChatGPT Plus, Pro, Business, Enterprise/Edu에 포함된다는 점, 한시적으로 Free/Go에도 포함된다는 점, CLI·IDE extension·app·web 사용 경로, 플러그인 제어, RBAC, Compliance API, Local/Cloud 사용 경계까지 한 번에 정리한다.

실사용 관점에서 중요한 이유: Codex의 최근 공식 변화는 새 버튼 추가보다 control plane 문서 정비에 가깝다. 실제 도입 판단은 기능 데모보다 어떤 플랜에서 무엇이 열리고, 어떤 surface에서 어떤 권한과 데이터 제어가 적용되는지에 더 좌우된다.
