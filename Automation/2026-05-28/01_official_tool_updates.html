Claude Code

Claude Code 2.1.152

확인 완료 공식 changelog · 2026-05-27

  • /code-review --fix가 리뷰 결과를 작업 트리에 적용하고 재사용, 단순화, 효율성 제안을 표면화한다. /simplify는 이 흐름을 호출하는 방향으로 정리됐다.
  • Skills와 slash command frontmatter에서 disallowed-tools를 지정해 특정 도구를 활성 세션에서 제거할 수 있게 됐다.
  • /reload-skills 명령과 SessionStart hook의 reloadSkills: true 응답으로 세션 재시작 없이 skill 디렉터리를 재스캔할 수 있다.
  • SessionStart hook이 세션 제목을 설정할 수 있고, MessageDisplay hook이 표시되는 assistant 메시지를 변환하거나 숨길 수 있다.

실사용 의미: Claude Code는 이번 릴리스에서 코드 리뷰 후 자동 수정, skill 재적재, hook 기반 표시/세션 제어처럼 에이전트 운영 표면을 넓혔다. 팀이 AGENTS/skills/hooks를 적극적으로 쓰는 경우, 권한과 컨텍스트 관리 규칙을 더 세밀하게 구성할 수 있다.

출처: Claude Code changelog

Cursor

Shared Canvases and /loop Skill

확인 완료 공식 changelog · 2026-05-20, 이번 3일 범위 밖

  • Cursor 3.5는 agent가 만든 보고서, 대시보드, 커스텀 인터페이스 같은 canvas를 팀과 공유하는 기능을 추가했다.
  • /loop skill은 프롬프트를 로컬 일정 또는 특정 결과가 나올 때까지 반복 실행하는 기능으로 설명된다.

실사용 의미: 이번 조사 기간 안의 신규 공식 업데이트는 확인되지 않았다. 다만 바로 직전의 3.5 업데이트는 Cursor가 IDE 보조 도구에서 팀 단위 agent artifact 관리 표면으로 이동하고 있음을 보여준다.

출처: Cursor changelog

OpenAI Codex

Appshots, Goal mode, remote computer use

확인 완료 공식 changelog · 2026-05-21, 이번 3일 범위 밖

  • Codex app의 Appshots는 macOS에서 앞쪽 앱 창의 스크린샷과 텍스트 컨텍스트를 Codex로 보내는 기능으로 안내됐다.
  • Goal mode는 Codex app, IDE extension, CLI에서 실험 기능이 아닌 일반 기능으로 제공된다고 명시됐다.
  • remote computer use는 Mac이 잠긴 뒤에도 Codex가 데스크톱 앱을 사용할 수 있게 하는 방향으로 설명됐다.

실사용 의미: 이번 3일 범위 안의 신규 공식 업데이트는 확인되지 않았다. 직전 업데이트의 핵심은 장시간 목표 수행, 앱 컨텍스트 캡처, 원격 컴퓨터 사용으로 Codex의 비동기 작업 수행 폭을 넓히는 것이다.
