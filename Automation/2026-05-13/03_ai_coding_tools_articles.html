Claude Code 관련

AI coders are carrying half-open laptops...

뉴스 기사Business Insider2026-05-12

Business Insider는 Claude Code, Codex, OpenCode 사용자가 agent 작업을 끊지 않기 위해 노트북을 반쯤 연 채 이동하는 사례를 다뤘다. 기사 속 사례들은 장시간 agent task가 개발자의 물리적 작업 습관까지 바꾸고 있음을 보여준다.

실무 시사점: agent 실행이 로컬 노트북 sleep/network 상태에 취약하다면 팀 차원의 원격 runner, background session, 작업 재개 전략이 필요하다.

신뢰성 메모: 뉴스 인터뷰 기사. 현상 묘사는 유용하지만 통계로 일반화하면 안 된다. 출처: Business Insider

Higher usage limits for Claude...

공식 글Anthropic2026-05-06

Anthropic은 Claude Code의 5시간 rate limits를 주요 유료 plan에서 두 배로 늘렸고 Pro/Max peak-hour 제한 축소를 제거했다고 밝혔다. 7일 창보다 하루 앞선 발표지만, 이번 주 Claude Code 기사 흐름을 해석하는 기준점이다.

실무 시사점: 더 긴 세션이 가능해지면, 사용량보다 리뷰/검증/중단 기준이 병목이 된다.

신뢰성 메모: 공식 발표. 출처: Anthropic

Agents for financial services

공식 글Anthropic2026-05-05

Anthropic은 금융 서비스용 agent templates를 Claude Cowork, Claude Code plugin, Claude Managed Agents cookbook 형태로 제공한다고 발표했다. strict 7일 창 밖이지만 이번 주 GitHub Trending의 anthropics/financial-services와 직접 연결된다.

실무 시사점: Claude Code는 범용 코딩 도구를 넘어 산업별 plugin/cookbook 배포 표면으로 확장 중이다.

신뢰성 메모: 공식 발표. 출처: Anthropic

Cursor 관련

Cursor in Microsoft Teams

공식 글Cursor2026-05-11

Cursor는 Teams channel에서 @Cursor를 호출해 cloud agent에게 작업을 위임하고, thread context를 읽은 뒤 구현 및 PR 생성을 수행할 수 있다고 설명한다.

실무 시사점: agent가 팀 대화 context를 읽고 repository를 자동 선택하면 편하지만, 잘못된 repo 선택과 권한 범위가 새 리스크가 된다.

신뢰성 메모: 공식 changelog. 출처: Cursor Changelog

Bugbot Effort Levels

공식 글Cursor2026-05-11

Cursor는 Bugbot PR review effort를 Default, High, Custom으로 조정할 수 있게 했다. 공식 수치로 default effort 평균 0.7 bugs/run, high effort 평균 0.95 bugs/run을 제시했다.

실무 시사점: AI review도 cost/performance profile을 분리해야 한다. 모든 PR이 아니라 위험 PR에 high effort를 쓰는 정책이 합리적이다.

신뢰성 메모: 공식 changelog. 출처: Cursor Changelog

Claude Code vs. Codex vs. Cursor vs. GitHub Copilot

비교 리뷰Built In2026-05-12

Built In은 네 도구를 동일한 TDD loop로 비교했다고 설명하며 Cursor를 IDE 기반 모델 전환에는 좋지만 UI clutter가 있다고 평가했다. 또한 AI 변경의 43%가 debugging을 필요로 했다고 요약했다.

실무 시사점: Cursor 도입 효과는 "더 많은 생성"보다 TDD, reviewer skill, PR 품질 게이트와 함께 봐야 한다.

신뢰성 메모: 비교 리뷰. 방법론은 유용하지만 표본/재현성은 별도 확인 필요. 출처: Built In
