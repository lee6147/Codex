Claude Code

Claude Opus 4.8 및 dynamic workflows

확인 완료 발표일: 2026-05-28 · 출처: Anthropic 공식 발표, Claude Code GitHub 릴리스

  • Opus 4.8은 같은 가격으로 제공되며, Claude Code에는 대규모 문제를 다루는 dynamic workflows 연구 미리보기가 추가되었습니다.
  • Anthropic은 Claude Code가 수백 개 병렬 subagent를 실행하고 결과를 검증한 뒤 보고할 수 있다고 설명합니다.
  • v2.1.154 릴리스는 Opus 4.8, dynamic workflows, fast mode 가격 조정, lean system prompt 기본화, `/simplify` 의미 변경을 포함합니다.

v2.1.156 hotfix

확인 완료 릴리스일: 2026-05-29 · 출처: GitHub Releases

  • Opus 4.8 사용 중 thinking block이 수정되어 API 오류가 나는 문제를 수정했습니다.
  • 2026-05-28 v2.1.153과 v2.1.154가 연속으로 나온 뒤 다음 날 안정화 패치가 붙은 흐름입니다.
  • 실무 관점에서는 대규모 병렬 워크플로우를 켜기 전에 버전 고정과 changelog 확인이 더 중요해졌습니다.

OpenAI Codex

Codex 공식 최신 항목

확인 완료 공식 Codex 항목: 2026-05-21 · 출처: OpenAI Help Center
