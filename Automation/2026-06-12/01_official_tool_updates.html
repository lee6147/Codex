Claude Code

2.1.173 / 2.1.172 계열 안정성 업데이트

공식 changelog 날짜 확인 불가

공식 changelog에는 Fable 5 모델명 정규화, Windows sandbox 경고 수정, 하위 에이전트의 재귀적 하위 에이전트 생성, Bedrock 리전 탐색, 플러그인 marketplace 검색, 1M context 세션 compaction, 백그라운드 에이전트 권한/상태 표시 수정 등이 포함됐다.

실무 관점에서는 대화가 길거나 여러 에이전트를 병렬로 쓰는 팀에서 발생하는 상태 불일치, 권한 상속, Windows 콘솔 이슈를 줄이는 성격이 강하다. 다만 원문 changelog가 날짜를 직접 노출하지 않아 최근 3일 공식 업데이트로 단정하지 않는다.

Cursor

Bugbot 리뷰가 더 빠르고 저렴해졌다는 공식 발표

공식 changelog 2026-06-10 리뷰 자동화

Cursor는 Bugbot 평균 리뷰 시간이 약 5분에서 약 90초로 줄고, 평균 발견 버그 수가 0.56에서 0.62로 증가했으며, 실행 비용도 약 22% 감소했다고 밝혔다. 이 개선은 Composer 2.5 학습 진전에 기반하며, 모델 block list를 존중한다고 설명했다.

새 `/review` 흐름은 푸시 전 Bugbot 및 Security Review를 실행하게 하고, 이후 동일 diff로 PR을 열면 GitHub/GitLab의 Bugbot이 이미 검토된 diff로 인식해 중복 리뷰를 피한다. 실무적으로는 리뷰 피드백을 PR 이후가 아니라 로컬/사전 단계로 앞당기는 변화다.

OpenAI Codex

Codex app 26.608: 마이그레이션 흐름과 플러그인 화면 개선

공식 changelog 2026-06-09 데스크톱 앱

Codex app 26.608은 Claude Code 및 Claude Cowork에서 지원되는 설정을 가져오는 "Migrate to Codex" 흐름을 추가했고, 플러그인 화면을 탭/마켓플레이스/카테고리 필터 중심으로 재구성했다. Settings 검색 범위도 Git 및 pets 등 더 많은 패널로 확장됐다.

성능 및 버그 수정에는 좁은 레이아웃에서 goal timer 겹침 수정, active goal 진행 중 읽지 않은 알림 감소, review diff 순서 안정화, Windows 10 등 반투명 backdrop 미지원 환경의 렌더링 개선이 포함됐다.

발표일: 2026-06-09 | 출처명: OpenAI Codex Changelog | 확인 상태: 확인 완료

Codex CLI 0.139.0: 코드 모드 웹 검색과 MCP 호환성 개선

공식 changelog 2026-06-09 CLI

Codex CLI 0.139.0은 코드 모드에서 독립 웹 검색을 직접 호출할 수 있게 했고, 중첩 JavaScript 도구 호출에서도 plaintext 검색 결과를 받을 수 있다고 설명한다. 또한 tool/connector input schema에서 `oneOf`와 `allOf` 보존을 개선해 복잡한 MCP 도구와의 호환성을 넓혔다.

추가로 `codex doctor`의 로컬 리포트 환경 정보 보강, 플러그인 marketplace JSON 출력 개선, `resume --last`/`fork --last` 인자 처리 수정, MCP startup warning 범위 수정, 이미지 편집 파일 경로 처리 개선 등이 포함됐다.

발표일: 2026-06-09 | 출처명: OpenAI Codex Changelog | 확인 상태: 확인 완료
