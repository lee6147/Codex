Cursor

Cursor SDK 공개

도구: Cursor 확인 완료 발표일 2026-04-29

Cursor는 2026-04-29 공식 changelog에서 Cursor SDK를 공개했다. TypeScript 모듈로 에이전트를 프로그래밍적으로 호출할 수 있고, 로컬 또는 Cursor Isolated VM에서 실행할 수 있으며, Cloud Agents API도 durable runs 기준으로 재구성했다고 설명한다.

문맥 보강 확인 완료 발표일 2026-04-24

Cursor 3.2는 /multitask, worktrees, multi-root workspaces를 전면에 세웠다. UI에서 병렬 작업과 여러 저장소 문맥을 다루는 철학이 먼저 나왔고, 이번 SDK는 그 철학을 코드 호출면으로 확장한다.
