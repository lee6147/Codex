Cursor SDK 공개

게시일 2026-04-29 매체 Cursor 공식 글

Cursor는 SDK 공개 글에서 TypeScript 기반 호출면, 로컬 또는 Isolated VM 실행, Cloud Agents API의 durable runs를 한 번에 설명했다. 핵심은 “편집기 내 기능”을 넘어 외부 서비스와 파이프라인에서 Cursor agent를 호출하게 만든다는 점이다.
