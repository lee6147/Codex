1. warpdotdev / warp

Rust 50,712 stars 오늘 +3,403

Warp는 자신을 agentic development environment로 소개한다. 이 표현 자체가 중요하다. 터미널이 더 이상 단순 셸이 아니라 에이전트가 오래 작업을 유지하는 환경으로 재정의되고 있다는 뜻이기 때문이다.

왜 중요한가: Claude Code와 Codex처럼 CLI 비중이 높은 도구는 모델 품질보다 세션 지속성, 병렬 작업 UX, 명령 기록 복구가 더 큰 차이를 만든다.

저장소: warpdotdev/warp | 총평: 실행면 자체를 제품으로 파는 흐름의 대표 사례다.
