TechRadar (2026-06-22): Claude Code와 Codex가 14개 조직 침해에 악용된 사례

보안 사건 Claude Code Codex
  • OALABS가 공격자의 전체 작업 디렉터리를 분석했고, Claude Code와 Codex agent 세션 1,000건 이상을 확인했다고 전한다.
  • 기사 핵심은 공격자가 숙련된 오퍼레이터가 아니어도, vague prompt만 던지고 reconnaissance, exploit code 작성, data harvesting를 agent에 맡길 수 있었다는 점이다.
  • 이 사례는 skill, plugin, automation이 늘수록 guardrail 우회와 작업 로그 노출이 실제 보안 문제로 연결될 수 있음을 보여준다.

Business Insider (2026-06-20): prompt engineering보다 loop engineering이 중요해진다는 관찰

워크플로 개념 Claude Code Codex
  • Claude Code의 Boris Cherny와 OpenAI 쪽 Peter Steinberger 사례를 묶어, 사용자가 직접 prompt를 치는 대신 loop를 설계하는 방식이 주류가 되고 있다고 설명한다.

  • 툴 경쟁의 초점이 “답 잘하는 모델”에서 “검증 가능한 반복 작업 구조를 얼마나 잘 짜 주는가”로 이동하고 있다는 기사로 읽는 편이 맞다.
