Claude Code

v2.1.118 공개 릴리스

공식 릴리스 2026-04-23 GitHub Releases

Anthropic는 2026-04-23에 Claude Code v2.1.118을 공개했다. 이번 릴리스는 단순 버그픽스보다 실사용 작업 흐름을 다듬는 성격이 강했다. Vim visual mode, /usage 통합, 커스텀 테마 전환, MCP tool hook 호출, WSL 정책 상속, 재인증/토큰 갱신 안정화가 한 번에 묶였다.

업무 관점에서는 “장시간 세션 + 플러그인/MCP + Windows/WSL” 환경을 실제 기본 시나리오로 본다는 신호다. 특히 /doctor, /plugin, OAuth 관련 항목이 많아졌다는 점은 확장 생태계가 빨라지는 대신 운영 난이도도 같이 올라간다는 뜻이다.

v2.1.117 공개 릴리스

공식 릴리스 2026-04-22 GitHub Releases

바로 전날인 2026-04-22에는 v2.1.117이 배포됐다. 포인트는 포크된 서브에이전트 활성화, --agent 경로에서의 MCP 서버 로딩, /resume 요약 개선, 플러그인 의존성 자동 복구, OpenTelemetry 확장, Windows where.exe 캐시, 장문 세션/프록시/토큰 만료 버그 수정이다.

OpenAI Codex

Codex 접근 범위 문서 갱신

공식 문서 2026-04-23 반영 2026-04-24 조회

OpenAI Help Center의 Using Codex with your ChatGPT plan 문서는 2026-04-24 조회 시 최근 갱신 상태였다. 문서상 핵심은 Codex가 Plus, Pro, Business, Enterprise/Edu뿐 아니라 한시적으로 Free와 Go에도 포함되며, Pro 구독에는 2배 rate limit이 적용된다는 점이다.

또한 Codex app, CLI, IDE extension, web을 하나의 접근 체계로 묶고, 플러그인/App 통제와 RBAC, Compliance API, Codex Local/Cloud 권한 분리를 더 명확히 적어 두었다. 이건 Codex가 단일 기능보다 “조직 배포 표면”으로 정리되고 있음을 뜻한다.

Codex rate card 전면 확대

공식 문서 2026-04-23 2026-04-24 재확인

Codex rate card 문서는 2026-04-24 조회 시 매우 최근 갱신 상태였고, 본문에는 2026-04-23부터 기존 ChatGPT Enterprise뿐 아니라 Edu, Health, Gov, ChatGPT for Teachers까지 토큰 기반 요금 체계 적용 범위를 넓혔다고 적혀 있다.

즉 OpenAI는 Codex 기능 추가만이 아니라 과금 모델을 더 넓은 조직군으로 밀어 넣는 단계에 들어갔다. 모델별 토큰 단가와 Fast mode 비용 차이가 문서에 더 직접적으로 드러나면서, 이제 Codex 도입 평가는 성능 비교만으로 끝나지 않고 예산 통제까지 같이 봐야 한다.
