Cursor

Design Mode Improvements

2026-06-05공식 글

Cursor 브라우저의 Design Mode에서 다중 요소 선택과 음성 입력이 강화됐다. 사용자는 브라우저에서 여러 UI 요소를 직접 선택하고, agent에게 일괄 정렬·삭제·수정 요청을 할 수 있다. 음성 입력은 agent 실행 중에도 다음 변경을 queue할 수 있게 한다.

실무 시사점: UI 구현 작업에서 텍스트 지시만으로 생기는 모호함을 줄인다. 프런트엔드 팀은 screenshot 기반 피드백을 코드 수정 루프로 바로 연결할 수 있다.

Custom stores, custom tools, and auto-review for the Cursor SDK

2026-06-04공식 글

Cursor SDK가 custom tools, auto-review, JSONL/custom stores, nested subagents, run correlation, bundled ripgrep 등 production scripting과 CI 사용성을 보강했다. 특히 local agent tool call을 auto-review classifier로 라우팅하는 기능은 headless 실행 위험을 낮춘다.

실무 시사점: Cursor가 IDE 도구를 넘어 SDK 기반 자동화 runtime으로 확장되고 있다. CI나 내부 agent script에 붙일 때 persistence와 permission 정책을 먼저 설계해야 한다.
