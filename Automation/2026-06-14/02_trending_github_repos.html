트렌드 해석

  • 공식 vendor CLI가 커졌다. Claude Code와 Codex는 단순 helper가 아니라 permission, background run, browser, plugin, mobile, remote session을 품는 운영 환경으로 이동 중이다.
  • 오픈소스 도구는 모델 독립성과 self-hosting을 무기로 삼지만, archive된 인기 프로젝트가 보여주듯 지속 유지보수가 핵심 리스크다.
  • Cursor 사용자는 IDE/Design Mode 흐름, Claude Code 사용자는 장시간 agent와 workflow, Codex 사용자는 앱·CLI·브라우저 통합을 비교하면 된다.
