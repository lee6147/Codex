Claude Code

v2.1.119 릴리즈 공개

공식 릴리즈 2026-04-23 GitHub Releases

Anthropic는 2026-04-23 23:24에 Claude Code v2.1.119를 공개했다. 이번 릴리즈에는 /config 설정 영속화, PowerShell 자동 승인 동작 개선, 서브에이전트와 MCP 재구성 병렬 연결, worktree 재사용 문제 수정, Windows 관련 경고 정리 등 실제 작업 흐름을 매끈하게 만드는 항목이 다수 들어갔다.

최근 품질 저하 보고 포스트모템 공개

공식 엔지니어링 글 2026-04-23 품질 회복

Anthropic Engineering은 2026-04-23에 최근 Claude Code 품질 저하 보고에 대한 포스트모템을 공개했다. 원인은 기본 reasoning effort를 high에서 medium으로 낮춘 결정, 유휴 세션의 사고 이력을 과도하게 지우는 캐시 최적화 버그, 그리고 2026-04-16에 넣었던 과도한 길이 제한 프롬프트였다.

문서에는 세 문제가 모두 2026-04-20까지 해결됐고, 더 넓은 eval, prompt 변경 통제, 내부 사용 빌드와 공개 빌드 간 간극 축소, 추가 저장소 맥락을 활용한 Code Review 개선을 하겠다고 적혀 있다. 사용량 한도 리셋까지 동반된 점은 단순 해명보다 운영 신뢰 회복 성격이 강하다.

Cursor

Cursor 3.2: Multitask, Worktrees, Multi-root Workspaces

공식 changelog 2026-04-24 Cursor 3.2

Cursor changelog의 최신 항목은 2026-04-24 공개된 Multitask, Worktrees, and Multi-root Workspaces다. 핵심은 /multitask를 통한 async subagents 병렬화, Agents Window의 worktree 개선, 그리고 하나의 세션이 여러 폴더를 겨냥할 수 있는 multi-root workspace 도입이다.

이 릴리즈는 Cursor가 더 이상 “IDE 안의 한 명짜리 보조자”가 아니라 여러 작업을 동시에 오케스트레이션하는 에이전트 콘솔을 지향한다는 점을 다시 확인시킨다. 복수 저장소와 병렬 worktree를 다루는 팀에게 특히 의미가 크다.

앱 안정성 엔지니어링 공개

공식 연구 글 2026-04-21 안정성

Cursor Research는 2026-04-21에 Keeping the Cursor app stable를 공개했다. 글에 따르면 late-February 대비 OOM-per-session은 80% 감소했고, 2026-03-01 이후 OOM-per-request는 73% 감소했다. 원인 분석에는 crash telemetry, crash watcher, heap snapshot, allocation profiling, Bugbot rules, automated rollback 등이 동원됐다.

멀티에이전트와 브라우저 사용 기능이 늘수록 앱 메모리 안정성이 핵심 경쟁력이 된다는 점을 공식적으로 보여준 문서다. Cursor가 기능 추가와 동시에 “앱이 하루 종일 버텨주는가”를 별도 연구 축으로 관리하고 있다는 신호로 읽힌다.

OpenAI Codex

Codex rate card 최신화

공식 도움말 2026-04-24 확인 요금 체계

OpenAI Help Center의 Codex rate card는 2026-04-24 시점 최신 버전으로 보였고, 본문에는 2026-04-23부터 기존 ChatGPT Enterprise뿐 아니라 Edu, Health, Gov, ChatGPT for Teachers까지 토큰 기반 Codex 가격표 적용 범위를 넓혔다고 적혀 있다.

핵심은 per-message 감각이 아니라 input, cached input, output token 기준으로 크레딧을 잡는 구조가 사실상 표준이 됐다는 점이다. Codex 평가는 이제 “잘 되느냐”보다 “어떤 모델과 fast mode를 쓰면 얼마가 드느냐”까지 같이 봐야 한다.

ChatGPT 플랜 기준 Codex 접근 문서 갱신

공식 도움말 2026-04-23 접근/권한

Using Codex with your ChatGPT plan 문서는 2026-04-23 기준으로 갱신돼 있었다. 문서에는 Codex가 Plus, Pro, Business, Enterprise/Edu에 포함되며, 한시적으로 Free와 Go에도 포함되고, 다른 플랜은 2배 rate limits를 제공한다고 정리돼 있다.
