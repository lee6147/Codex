요약 통계

GitHub Trending 페이지는 확인했지만 repository card가 텍스트 도구에 노출되지 않았다. 따라서 이 문서는 GitHub Search API에서 pushed:>=2026-06-04, 관련 topic, star 수 기준으로 조회한 저장소를 선별했다. 실제 Trending 랭킹이 필요한 경우 GitHub 웹 UI에서 원본 화면을 별도 확인해야 한다.

트렌드 해석

패턴관찰실무 시사점
컨텍스트 비용 절감 rtk, serena, claude-context처럼 token과 retrieval을 직접 다루는 저장소가 눈에 띈다. AI 코딩 도구의 병목은 모델 성능만이 아니라 필요한 코드만 읽히는 구조다.
에이전트 하네스 표준화 ECC, ruflo, kilocode는 agent 실행 방식, memory, skill, workflow를 제품화하려 한다. 팀은 도구 하나를 고르는 것보다 재현 가능한 agent 운영 규칙을 먼저 정해야 한다.
실행 환경 분리 daytona류 sandbox가 계속 중요해진다. 에이전트가 코드를 실행하고 패키지를 설치할수록 보안 경계와 비용 경계가 필요하다.
