Claude Code

Claude Code v2.1.132

공식 GitHub 릴리스확인 완료

Anthropic의 Claude Code 저장소 최신 릴리스는 Bash 하위 프로세스에 `CLAUDE_CODE_SESSION_ID`를 전달하고, 대체 화면 렌더러 비활성화 환경 변수를 추가했다. 외부 SIGINT, SSH 연결 종료, resume 중 문자열 손상, plan-mode 재개, fullscreen 복구 같은 터미널 운용 문제도 다수 수정했다.

실사용 관점: 장시간 터미널 세션, IDE 중단 버튼, tmux/SSH 환경에서의 복구 안정성이 중요해졌다. 로컬 자동화나 에이전트 하네스는 세션 ID를 로그와 훅에 연결하기 쉬워진다.

금융 서비스용 에이전트 템플릿

공식 발표Claude Code 플러그인

Anthropic은 금융 서비스 업무용 ready-to-run 에이전트 템플릿 10개를 발표했다. 피치북, KYC, 결산, 밸류에이션 검토 같은 업무가 Claude Cowork와 Claude Code 플러그인, Claude Managed Agents cookbook 형태로 제공된다.

실사용 관점: Claude Code가 범용 코딩 보조 도구를 넘어 도메인별 업무 플러그인 실행 표면으로 확장되고 있다. 규제 산업에서는 도구 권한, 감사 로그, 검토자 승인 흐름이 핵심 체크포인트다.

Cursor

Context Usage Breakdown

공식 ChangelogCursor 3.3

Cursor는 에이전트의 컨텍스트 사용량을 분해해서 볼 수 있는 기능을 공개했다. 규칙, skills, MCP, subagents가 컨텍스트를 얼마나 쓰는지 진단하는 용도다.

실사용 관점: Cursor 기반 팀은 성능 저하나 응답 품질 저하를 "모델 문제"로만 보지 않고, 규칙/도구/서브에이전트가 컨텍스트를 과하게 쓰는지 점검할 수 있다.

OpenAI Codex

Codex 0.129.0

공식 GitHub 릴리스확인 완료

OpenAI Codex 저장소의 0.129.0 릴리스는 TUI composer의 Vim 편집 모드, resume/fork picker 개선, raw scrollback, `/ide` 컨텍스트 주입, workspace-aware `/diff`, 상태줄 색상/PR/브랜치 요약, 플러그인 workspace sharing을 포함한다.

실사용 관점: Codex CLI/TUI가 단순 채팅 셸보다 실제 개발 터미널에 가까워지고 있다. 플러그인 공유와 워크스페이스 단위 운영 기능은 반복 자동화 하네스와 조직 내 설정 재사용에 직접 연결된다.
