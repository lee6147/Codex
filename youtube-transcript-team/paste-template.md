# Transcript Intake Template

아래 형식으로 붙여 넣으면 된다.

```text
[영상 제목]

[채널명]

[영상 길이]

[업로드 날짜]

[내가 궁금한 것]
- 예: 이 영상 진짜 볼 가치 있나?
- 예: 코덱스와 클로드코드 중 실제 워크플로우 차이를 잘 설명하나?
- 예: 실무에 바로 적용할 팁만 뽑아줘.

[스크립트 완전성]
- 전체 스크립트 / 일부만 있음 / 자동 생성 자막이라 부정확할 수 있음

[내 수준]
- 초보 / 실무자 / 팀 리드

[이전에 본 영상]
- 제목 또는 채널명 위주로 1~5개

[자주 보는 채널]
- 없으면 비워도 됨

[스크립트]
여기에 전체 스크립트를 붙여 넣기
```

선택 항목

- 내가 이미 아는 내용
- 내가 쓰는 도구: Claude Code / Codex / Cursor / VS Code / MCP / 터미널 위주
- 관심 포인트: 자동화, 멀티에이전트, 리팩터링, 리뷰, 레거시 현대화, 보안
- 원하는 추천 방식: 전체 시청 우선 / 필요한 구간만 / 액션 위주 / 연결 영상 추천 포함
