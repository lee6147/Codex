# Analyses Folder

이 폴더에는 유튜브 영상 분석 결과를 영상별로 나눠 저장한다.

## 폴더 규칙

- 영상 하나당 폴더 하나
- 폴더명 형식: `YYYY-MM-DD__channel__video-slug`
- 채널명과 제목은 ASCII 위주로 짧게 정리

## 기본 파일 구성

### 00-intake.md
- 영상 제목
- 채널명
- 영상 길이
- 업로드 날짜
- 내가 궁금한 것
- 내 수준
- 이전에 본 영상
- 원문 스크립트 또는 transcript 파일 목록

### 01-analysis.md
- 압축 모드 또는 풀 버전 분석 결과
- 직접 볼 가치
- 실행 가능성
- 검증 필요 주장

### 02-followup.md
- 바로 실행할 것
- 같이 보면 좋은 영상
- 다음 비교 후보
- 개인 메모

## 메모

- 급할 때는 `01-analysis.md`만 먼저 채워도 된다.
- 나중에 자세한 분석으로 확장해도 같은 폴더를 계속 사용한다.
- 원문이 길면 `transcript/` 폴더를 만들고 `transcript-part-01.md`부터 순서대로 쪼개서 저장한다.
